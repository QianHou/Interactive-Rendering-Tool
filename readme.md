# INTERACTIVE RENDERING TOOL

## 依赖

- Qt 5.5.1
- OpenGL

## 编译

- `cd interactive_rendering_tool`
- (首次编译)添加执行权限：`sudo chmod 777 toolmake`
- 代码编译：`./toolmake`

## 运行

- `cd interactive_rendering_tool`
- (首次运行)添加执行权限：`sudo chmod 777 toolrun`
- 代码运行：`./toolrun`

## 使用说明

- 渲染显示窗口：鼠标左键控制旋转，鼠标滚轮控制缩放
